## Getting Started

To get started and run the app, follow the commands below:



### Running the app on [http://localhost:3000](http://localhost:3000)

```sh
npm run dev
```

## Running Tests

All tests located in [./\_\_tests\_\_](./__tests__) directory.

### Unit Tests
```sh
npm run test:unit
```

### Integration Tests
```sh
npm run test:integration
```

### E2E Tests
```sh
npm run test:e2e
```
